-----<< Initial Setup >>-----
1. Install python using the included "python-3.8.0-amd64.exe" installer, and select "add to PATH" box.
2. Install git using the included "Git-2.34.0-64-bit.exe" installer.
3. After Python and Git are installed, Run the included "gitclone.bat" file.
4. Read the README.txt file in the newly created "DCSWaypoint" Folder to continue setup.
